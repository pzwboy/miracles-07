## 🚀 使用

1. 下载**本项目 [master.zip](https://github.com/pzwboy/miracles-07/archive/refs/heads/master.zip)**
2. 确保自身 PHP 环境在 7.1.0 及以上版本，并装有 CURL 拓展（否则一些功能无法正常使用）
3. 解压后，**将该主题目录改名为`Miracles`（M 大写）**
4. 将文件夹放入 Typecho 根目录下的`usr/themes/`
5. 到后台-外观-可用的主题中，启用主题
